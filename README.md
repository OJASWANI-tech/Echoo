# Echoo
📹 Echoo – Real-Time Video Calling Platform

🚀 Live Demo: https://echoo-eight.vercel.app/

Echoo is a modern video conferencing and collaboration platform that enables seamless communication through real-time video, chat, and meeting management. Built with a clean UI and secure authentication, it is designed for both productivity and simplicity.

✨ Features
📜 Meeting History
View complete records of past meetings
Includes timestamps, participants, and duration
Helps track activity and revisit sessions
👥 Attendee Tracking
See all participants for each meeting
Useful for collaboration tracking and attendance monitoring
🔐 Google Authentication
Secure and fast login using Google OAuth
No need to remember passwords
💬 Real-Time Video & Chat
High-quality video and audio communication
Integrated real-time messaging
Smooth and responsive experience
🎨 Simple & Intuitive UI
Clean and minimal interface
Easy navigation
Designed for all users (technical & non-technical)
📝 Editable Profile
Update your name and profile picture
Customize your user preferences anytime
